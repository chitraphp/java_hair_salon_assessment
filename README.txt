Date: August 22, 2015

Description
The owner is able to add a list of the stylists, and for each stylist, able to add clients who see that stylist. The stylists work independently, so each client only belongs to a single stylist.

Setup Instructions:
1.clone the application
2.Go to terminal and enter "gradle run"
3.Enter URL "localhost:4567" on any browser
